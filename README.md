## 📌 Overview

**Academic Paper Downloader** is a Python tool designed to automate the process of searching, extracting, downloading, and organizing academic papers.

The program retrieves papers from online sources, downloads them in batch, and saves each file using its corresponding paper title for better organization and accessibility.

---

## ✨ Features

* 🔍 Automatic extraction of academic papers
* ⬇️ Batch downloading
* 🏷️ Automatic file renaming based on paper titles
* 📂 Organized local storage
* ⚡ Lightweight and easy to use

---
